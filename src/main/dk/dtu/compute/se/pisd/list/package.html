This package is part of the course Project in Software Development at DTU Compute.

This package defines an interface for a {@see dk.dtu.compute.se.pisd.list.List} of {@see java.lang.Integer} values as well as a stub for an implementation of interface {@see dk.dtu.compute.se.pisd.list.List} based on array.

This interface and the implementations will be extended in later tutorials of the course.
