This is an example of how to use Java for implementing data types and testing them by JUnit test. This is part of the course Project in Software Development at DTU Compute.

The current example is a simple {@see dk.dtu.compute.se.pisd.list.List} of {@see java.lang.Integer} values. But, this will evolve to a more sophisticated version during the course.
